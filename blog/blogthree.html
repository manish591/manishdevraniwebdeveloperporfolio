It is called skeleton of the web page . It is used to layout the website. Hyper text means it is used to link to various documents in the world wide web thus make a connection in the web page. HTML uses various markup tags to layout HTML document. Like head tag, heading tag, paragraph tag etc.

Before We Begin

You can also see that above the html tag we have Doctype HTML and lang attribute. Let us explore them first.
DOCTYPE HTML tells Instruct document that which version to use in this document. The above tag tells browser to use HTML 5 in the document. If you didn't write this in your document it will get confused. It can't identify whhich version of code to expect. It can still parse code but also introduced bugs in your code sometimes.
lang="en" attribute used to identify the language of text content on the web. en here means to expect english language. There are other various code available.
Let's discuss what's inside head of the html document.

Title Tag

This tag is used to describe title of the whole document. Don't confuse it with h1 heading tag. H1 is present on the web page itself but title tag content present on tab section.

Linking to css and javascript

Having a separate CSS and Javscript file is must. This can make your code more beautiful and human readable. You can link your external CSS and Javscript file inside head using link and script tags respectively.
For Linking CSS you have to use link tag with href and rel attribute.
For Linking to Javscript you have to use script tag with source attribute.

Adding custom icons using favicon

You can also add favicons short for favorite icons on your web page using favicons. they appear right in front of title of your tab. They can improve design of website and make it more attractive.

Meta tags

Meta data is the data that describes the data of document. It won't show on web page but do a lot behind the scenes. It is useful for search engine optimization. It is used to describe content, author, description keyword related to the page. Let's explore them.

charset attribute of meta tag

The charset attribute of meta tag is used to specify the character encoding of the HTML document. It's value must be set to UTF-8, It can identify every charcter of any language. If it is not set to UTF-8 then it will not render certain language character properly.

Define keyword, author, description with meta tag

We can define keyword using name = "keyword", which will help in seo and generate more traffic. Similarly author and descrption gives info about document. Who has written this. If anyone need that information he can access it.
